Project Overview

This project analyzes 29,999 Android apps using their metadata and permission requests to determine their potential privacy risk level.

The dataset includes 184 columns, including system permissions, app category, ratings, prices, and more. Machine learning is used to classify apps into Low or High risk.

Techniques Used

  • Exploratory Data Analysis (EDA)
  • Permission-level feature engineering
  • Random Forest Classification
  • Hypothesis-driven privacy research
  • Model explainability through feature importance
